## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, solid shadow translation X/Y |
| sm | 8px | Thick borders, compact elements |
| md | 16px | Default element spacing |
| lg | 24px | Card paddings |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions: Drop shadows are solid, unblurred color translations (e.g. `box-shadow: 4px 4px 0px #000`). All `border-radius` values are explicitly `0px`.

## Color

Neo-brutalism demands stark contrasts and pure hues. 

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#FFFDF9` | Backgrounds, surfaces (off-white for eye comfort) |
| Secondary (30%) | `#E8E8E8` | Sidebar overlays, disabled cards |
| Accent (10%) | `#FF3B30` | Core action CTAs (Vibrant Red) |
| Brutalist Stroke | `#000000` | Pure black for card borders (2-4px), drop shadows |
| Highlight | `#FFD60A` | Highlighting critical homily excerpts |

Accent reserved for: Search buttons, pagination active blocks, returning to the list view. Never generic texts.
